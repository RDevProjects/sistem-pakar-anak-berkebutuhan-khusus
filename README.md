## Sistem Pakar Anak Berkebutuhan Khusus

Proyek ini adalah sistem pakar yang dirancang untuk membantu mengidentifikasi kebutuhan khusus anak berdasarkan gejala yang diinputkan oleh pengguna. Sistem ini menggunakan metode berbasis aturan untuk memberikan rekomendasi yang sesuai.

### Fitur Utama

-   Input gejala anak secara interaktif.
-   Hasil analisis dan rekomendasi kebutuhan khusus.
-   Antarmuka pengguna yang ramah dan mudah digunakan.
-   Basis data gejala dan rekomendasi yang dapat diperbarui.

### Teknologi yang Digunakan

-   Laravel sebagai kerangka kerja backend.
-   Blade untuk templating frontend.
-   MySQL untuk basis data.

### Cara Menggunakan

1. Clone repositori ini ke komputer Anda.
2. Jalankan perintah `composer install` untuk menginstal dependensi.
3. Konfigurasikan file `.env` untuk koneksi basis data.
4. Jalankan migrasi dengan perintah `php artisan migrate`.
5. Mulai server lokal dengan perintah `php artisan serve`.
